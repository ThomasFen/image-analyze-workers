# TFLite Wasm for White Box Cartoonization
## Build wasm
```
$ npm run build_docker
$ npm run start_docker
$ npm run build_wasm
$ npm run build_wasm_simd
$ npm run build_wasm_for_safari
$ npm run stop_docker
```
## Note
Safari SIMD is not supported. 

```
invalid opcode 253
```
